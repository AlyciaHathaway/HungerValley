### 基本概念
![enter description here][1]
### flex container 的属性

 - flex-direction：方向
 - flex-wrap：换行
 - flex-flow：上面两个的简写
 - justify-content：主轴方向对齐方式
 - align-items：侧轴对齐方式（高度不能写死，用内容来撑开）
 - align-content：多行 / 列内容对齐方式（用的较少）


### flex items 的属性

 - flex-grow：增长比例（空间过多时）
 - flex-shrink：收缩比例（空间不够时）
 - flex-basis：默认大小（一般不用）
 - flex：上面三个的缩写
 - order：顺序（代替双飞翼）
 - align-self：自身的对齐方式（高度不能写死）
